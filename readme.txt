== Description ==
<p>
Show welcome pop-up box on page. It is great-looking and responsive!. 
Visit [demo](https://walexconcepts.com/wordpress/).
</p>

Free Version features:
<p>
<ul>
<li>* Show box to everyone-all pages</li>
<li>* Insert / edit embedded media like YouTube videos,..</li>
<li>* Text can be completely formatted.</li>
<li>* Insert / edit image</li>
<li>* Insert / edit table</li>
<li>* HTML source editor for update.</li>
<li>* Coding skills NOT required.</li>
<li>* Theme modifications NOT required.</li>
<li>* Add back office functionality.</li>
<li>* No dependency on 3rd party service.</li>
<li>* Use 'Welcome Popup Box' menu at admin page for configuration.</li>
</ul>
</p>

Premium Features:
<p>
<ul>
<li>* All of te above free version features. </li>
<li>* Click on switch button to Enable / Disable box</li>
<li>* Show box just to homepage, logged users, or everyone.</li>
<li>* Insert emotions icon</li>
<li>* Insert predefined template content.</li>
<li>* Previev content before updating.</li>
<li>* Change text color</li>
<li>* Change background color</li>
<li>* Insert special character</li>
</ul>
</p>

== Installation ==
<p>
<ul>
<li>1. Upload files of Welcome Popup Box to the `/wp-content/plugins/`</li> 
<li>2. Activate the plugin through the 'Plugins' menu in WordPress</li>
<li>3. At backend, click 'Welcome Popup Box' menu in WordPress to open Welcome Popup Box admin page.</li>
<li>4. No Shotcode or Widget required</li>
</ul>
</p>
